Shell commands/programs:
-whoami: current user name
-cd: changes directory - + ".." to move back to parent directory
-pwd: prints current working directory
-ls: list 
  • -l for  long listing format 
-man: manual
-mkdir: make new directory
-rmdir: remove directory
-touch: make new file
-rm: remove
  • -r = recursive
  • -i = interactive, will prmopt you to answer
-curl: "cat url" - transfers a url
-unzip: use for .zip files
-du: disk usage
-mv: move
-cp: copy
  • -r = recursive, use for dirrectories
-cat: concatenate
-wc: "word count" - also can count lines, characters, etc.
-sort
-head: print first n lines
-tail print last 10 lines
-echo
-tr: trim
  • -s = "squeeze"
-uniq
-history
  • -c = count
-less: text viewer, similar to cat
-nano: text editor
-clear: clear window

def center(data):

    [[ 0  1]
     [ 0 -1]]
